## Project installation

- **Install the vendor by [composer](https://getcomposer.org/)**

```
    composer update
```

- **Install the node_modules by [npm](https://nodejs.org/en/)**

```
    npm i
```

- **Install a package for JS indentation (pre-commit)**

```
    npm i prettier
```
