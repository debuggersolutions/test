## Project installation

Command to install the vendor folder

```
- **[composer](https://getcomposer.org/) update**
```

Install the node_modules folder

```
- **[npm](https://nodejs.org/en/) i**
```

Install a package for JS indentation (pre-commit)

```
- **npm i prettier**
```
